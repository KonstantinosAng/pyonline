# PyOnline

Online python playground using [Anaconda's pyscript](https://pyscript.net/), [reactJS](https://reactjs.org/) and [tailwind css](https://tailwindcss.com).

## Demo

<div align="center">
  <h1><a href="https://pyonline.constantine.dev"> DEMO </a></h1>
</div>

## Installation

```bash
yarn install

or

npm install
```

## Run

```bash
yarn start

or

npm start
```
